# Инструменты

Здесь лежат инструменты конвертации ресурсов.

Планируемые инструменты:

- Извлечение оригинальных паков ресурсов в нормализованные PNG/WAV/data-файлы.
- Конвертация тайлов карты и UI-графики в форматы bitmap для FT812.
- Разрезание крупных ресурсов на 16K SPG-страницы.
- Генерация assembly include-файлов с адресами и размерами ресурсов.
