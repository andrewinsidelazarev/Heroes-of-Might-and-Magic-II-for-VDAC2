# Open HMM2 TS-Config / VDAC2 Port

Стартовый проект порта open HMM2 / Heroes of Might and Magic II style engine для TS-Config с VDAC2 / FT812.

Цель проекта: перенести игровую модель, карту приключений, городские экраны, бой и UI в формат, пригодный для 640x480 вывода через FT812, с Z80-ядром, страничной упаковкой SPG и отдельным конвейером подготовки ассетов.

## Текущий статус

- Создан каркас проекта.
- Заведена базовая структура исходников `Source/ASM`.
- Заведены каталоги ассетов `Assets/Original` и `Assets/Converted`.
- Добавлен минимальный `build.cmd` и `spgbld_vdac2.ini`.
- Добавлены документы по архитектуре и карте памяти.
- Добавлен `Source/Tools/agg_tools.py` для аудита AGG и извлечения XMI-музыки.

Это не готовый порт игры. Это стартовая база, от которой можно последовательно переносить подсистемы HMM2.

## Структура

```text
Assets/
  Original/      исходные графика, карты, звуки, данные
  Converted/     бинарные данные для SPG/FT812
Build/           выход сборки
Docs/            проектная документация
Source/
  ASM/           Z80/TS-Config/FT812 код
  Tools/         конвертеры ассетов
```

## Сборка

По умолчанию `build.cmd` ищет инструменты в переменных окружения:

```bat
set SJASMPLUS=C:\z80\tsconf_project\exe\sjasmplus\sjasmplus.exe
set SPGBLD=C:\z80\tsconf_project\exe\spgbld\spgbld.exe
build.cmd
```

Если переменные не заданы, скрипт пробует типовые пути из TS-Config SDK.
