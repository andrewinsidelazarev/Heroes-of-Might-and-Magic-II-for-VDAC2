# Heroes of Might and Magic II for ZX Evolution / TS-Config / VDAC2

Играбельный порт Heroes of Might and Magic II для ZX Evolution с TS-Config и
VDAC2/FT812. Проект использует Z80-ядро, страничную упаковку SPG, потоковую
загрузку сцен из PAK-файлов и build-time конвертацию оригинальных ресурсов HMM2.

Текущая опорная версия: [v020 от 2026-07-16](Pre-releases/v020-2026-07-16-adventure-ui-battle-ai-reference/REFERENCE.md).
Она соответствует успешно собранному проекту с исходниками, Build, Docs и Music.

## Реализовано

- Подключена реальная локальная TSLib с FT812 и TS-Config include.
- Работает аппаратный вывод VDAC2/FT812: физический режим 1024×768, логическая
  игровая система координат 640×480, VSync и DMA/CMD-конвейер.
- Зафиксирован и используется конвейер игровых данных: карты MX2/MP2, terrain,
  объекты, герои, города, армии, монстры, бой и сценовые PAK-пакеты.
- Реализованы конвертеры AGG, ICN, TIL, карт, палитровой графики, атласов,
  анимаций и FT812-friendly форматов.
- Реализована карта приключений: реальная карта и ресурсы HMM2, прокрутка,
  мини-карта, fog of war, анимация воды и объектов, поиск пути, маршруты,
  перемещение и полная анимация героя по направлениям.
- Работают игровые объекты, подбор ресурсов, очки движения, завершение хода,
  смена дня, вражеский герой и переход в бой при столкновении.
- Реализованы главное меню, New Game, выбор сценария и экран рекордов.
- Реализован экран города: здания, подсказки, найм существ, экономика,
  гарнизон, таверна, рынок и колодец.
- Реализован боевой экран: движение, ближняя и дальняя атака, анимации,
  снаряды, курсоры, статусные строки, окна помощи, настройки скорости,
  AI противника и итог победы/поражения.
- Работает единый ввод с клавиатуры, Kempston Joystick и Kempston Mouse.
- Музыка `MIDIxxxx.XMI` конвертируется через MIDI в поток событий и
  воспроизводится аппаратным SAM2695 MIDI-синтезатором ZX Evolution.
- Сцены и данные загружаются с SD из `HMM2MENU.PAK`, `HMM2TOWN.PAK`,
  `HMM2BATL.PAK`, `HMM2HISC.PAK`, `HMM2MAP.PAK` и `HMM2SCN.PAK`.

Проект продолжает развиваться в сторону более полного соответствия оригинальной
HMM2/OpenHMM2. Это уже не стартовый каркас: основные игровые экраны и сквозной
playable-сценарий реализованы, но полный объём оригинальной игры ещё не перенесён.

## Сборка

Требуются Python 3, `sjasmplus` и `spgbld`. Пути к инструментам можно передать
через переменные окружения:

```bat
set SJASMPLUS=C:\z80\tsconf_project\exe\sjasmplus\sjasmplus.exe
set SPGBLD=C:\z80\tsconf_project\exe\spgbld\spgbld.exe
build.cmd
```

Скрипт конвертирует игровые данные, собирает Z80-код и scene overlays, создаёт
SPG/PAK-файлы и проверяет лимиты RAM_G, RAM_DL и FT812 display list.

Основной результат:

```text
Build/hmm2_vdac2.spg
```

В v020 проверено:

```text
sjasmplus: 0 errors, 0 warnings
Core split: p05=12288 bytes, p06=12210 bytes
RAM_DL max frame=7892/8192
RAM_G calculator: OK
SPG build: OK
FT812/RAM_G/DL verify: OK
```

Оригинальные игровые ресурсы не распространяются в репозитории. Для локальной
сборки требуются собственные файлы Heroes of Might and Magic II в
`Assets/Original`.

```text
Docs/             архитектура, карта памяти и описание подсистем

```
