# Pre-releases

Каталог опорных версий проекта Heroes of Might and Magic II for VDAC2.

Каждый снимок хранится в отдельном каталоге и содержит:

- архив исходников в структуре проекта;
- собранный образ `hmm2_vdac2.spg`;
- файл отладочных символов `hmm2.sym`;
- описание исходного Git-коммита и результатов проверки;
- SHA-256 контрольные суммы.
